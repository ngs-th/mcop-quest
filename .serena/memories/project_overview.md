# MCOP Quest - Project Overview

## Project Purpose
MCOP Quest เป็น Internal Gamification Dashboard ที่แปลงข้อมูลการพัฒนาโปรเจกต์ MCOP ให้กลายเป็นเกม RPG ธีมยุค 90s เพื่อสร้างแรงจูงใจให้ทีม พร้อมแสดงภาพรวมความคืบหน้าของโปรเจกต์

## Core Metaphor Mapping
| Project Concept | Game Concept |
|-----------------|-------------|
| System | City |
| Epic / Flow | Boss |
| Story | Boss Phase |
| Task | Minion |
| Bug | Demon Reinforcement |
| Project Completion | Demon King Defeated |

## User Roles
- **Business Team:** Support / Item Crafter
- **UX/UI Team:** Equipment Crafter
- **Backend Team:** Mage / Ranger (Backline DPS)
- **Frontend Team:** Warrior / Tank (Frontline)

## Key Features
- Google OAuth2 Authentication
- Real-time sync from Google Sheets
- World Map visualization (System = City)
- City View (Epic/Flow = Boss)
- Hero Dashboard (Personal View)
- Team Camp (Team View)
- Reward Shop
- Activity Log
